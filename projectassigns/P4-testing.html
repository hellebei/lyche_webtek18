Testing dokument

To get both qualitative and quantitative test results we created an usability test and a Google Form (link here) to test our web page. We wanted to test different aspects of the page's user-friendliness and usability. We sent the Form to friends, fellow students and family with a note saying that the page is aimed at Google Chrome, and recommended them to use this web browser (this is included in the form as well). For the live testing we developed a semistructured usability test with some warm up-questions, a couple of tasks to understand the users conseptual model of the web page and to test the navigation structure. We then asked them about the general layout and if they had other comments. Here's the document we used for the live testing:
